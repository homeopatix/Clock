# Installation (LOTRO)

## Pré-requis
- LOTRO installé
- Accès au dossier Plugins (Documents / Lord of the Rings Online)

## Installation
1. Télécharger le repo (Code → Download ZIP) ou cloner.
2. Copier le dossier du plugin au bon emplacement LOTRO (Plugins).
3. Lancer LOTRO.
4. Charger le plugin en jeu selon tes habitudes (Plugin Manager).

## Mise à jour
- Remplacer les fichiers du plugin par la nouvelle version.
- Conserver tes données si elles sont stockées côté LOTRO.

> Note: les chemins exacts peuvent varier selon la langue/OS. Si tu veux, je te fais la section “Windows FR/EN” précise selon ton installation.
