# Dépannage

## Le plugin ne se charge pas
- Vérifie l’emplacement dans le dossier Plugins.
- Vérifie que les dossiers du plugin n’ont pas été imbriqués après extraction ZIP (ex: AltHolic-main/AltHolic/...).

## Je ne vois pas l’icône
- Utilise `/Alt icon` pour la réafficher.
- Vérifie qu’elle n’est pas hors écran (ALT pour la déplacer).

## Les persos sur serveurs différents ont le même nom
- Attention : si plusieurs personnages ont exactement le même nom sur différents serveurs, certaines fonctions peuvent ne pas fonctionner.
